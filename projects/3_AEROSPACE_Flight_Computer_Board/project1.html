Flight Computer Board

1 - Project Description :

Flight Computer is on board 9 axis gyroscope + accelerometer (MPU-6050) to calculate the IMU (inertial measurement unit) of the object using PIC MicroController (PIC16F876A) for calculations and USB using uart_to_usb bridge (CP2102GM) for debugging and ICSP interface for programming/flashing code into MicroController and CAN BUS interface using CAN_Controller (MCP2515) and CAN_transeiver (TJA1050) for sharing the IMU informations to other CAN nodes (boards) .

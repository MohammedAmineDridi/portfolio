LaunchPad Computer Board

1 - Project Description :

LaunchPad Computer Board is a remote control system based on wifi protocol using (ESP_12S) for controlling the 4 clamps of the model rocket launchpad to start flying your model rocket remotly : ESP12S take your command for web/mobile app , then it send this command via 'AT+CMDS' to the MicroController (STM32L4) to open the 4 clamps by turning on the 4 DC motors .

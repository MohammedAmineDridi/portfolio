Smart Knee Rehabilitation Device

- The goal of this project is to create a smart knee rehabilitation device that restricts unexpected knee movement that can cause a knee problems and injuries .

- The solution is to create a device that sense the rotation angle of the knee, then define the threshold knee angle that cause a knee problem to create a control system algorithm that detect this unexpected wrong movement and try instantaneously to return back the knee to it's initial position (angle = 0->20°) using an industrial servo motor with high torque value controlled by current flow coming from the microcontroller to restrict those knee movements .

I - Concept & System Design Phase :

1 - Sensors :

a - Flex Sensor : to measue the knee's rotation angle in (°) .

b - Force Sensor : to measure the amount of force applied by the knee to the ground to calculate the amount of the motor torque needed to stop or restrict this unexpected movement .

Firing System Application

this project is : smart firing system based on threads like (slignshot or arrow game) , the idea is to design a smart firing system that shoot the bullet in specific target location whatever is the position of the src or the target , based on physics , dynamics and math calculations and system implementation using software and hardware , the system can choose the exact parameters values to shoot the bullet and hit the target correctly . Let's see how this system works ?

There is two essential parameters in controlling this system :

1 - angle of support of the bullet that control (TETHA angle) of bullet's projectile motion based on input params (DX/DY/L0/a/b/k/m) .

2 - angle of manivelle (ALPHA) that control the distance of elongation 'thread attraction' of ths system based of input params (r,l,n,l1/DL) .

1.4 - Conclusion & perspeective


advantage is : scalable architecture , the calculation process is in server side instead of MCU (relaxing the cpu of mcu for less calculation) .

if you like this project , i recommand you to do some opitimization tricks :


1 - using seperate mcu's (master : for control system via wifi) & mcu2 (slave) for controlling and driving the 5 servo motors of the system and all sensors and actuators .


2 - use low layer (driver developpement) , then (HAL) : to reduce the compiling and execution time (high compiling & execution time) .


3 - optimize the calculation of projectile angles by adding environmental factors such as (Friction, Air resistance, ...).


4 - target distance prediction (prediction to the next displacement of the target) => for optimize the calculation, using algorithms such as (Kalman filter) or artificial intelligence (AI) algorithms.
